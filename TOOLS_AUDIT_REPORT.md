## 1. 严重 Bug (🔴)

### 🔴 BUG-001: Shell 命令过滤过于激进，导致合法命令无法执行
**位置**: `base.py` 第 518-543 行 (`_validate_parameter_value`)

**问题描述**:
`shell_execute` 的参数验证将以下模式全部列为危险并拒绝执行：
- `&&`, `||`, `;` — 这导致**任何复合命令**都无法执行，例如 `npm install && npm run build`
- `$((`, `` ` ``, `${`, `$(` — 阻止所有 Shell 变量扩展和命令替换
- `base64`, `xxd`, `od` — 使用单词边界匹配，即使这些词出现在其他命令中（如 `code` 包含 `od`）也会误报

**影响**: Shell 工具几乎无法用于正常的构建、测试、部署流程，可用性极低。
